# Linux Monitoring Setup with Prometheus, Node Exporter, and Grafana

This repository provides a script to set up a Linux Monitoring system using Prometheus, Node Exporter, and Grafana. It automates the process of downloading, installing, and configuring Prometheus and Node Exporter services, as well as Grafana for visualization.

## Prerequisites

- A Linux-based system (Ubuntu/Debian recommended)
- Root or `sudo` privileges
- Internet access to download necessary files
